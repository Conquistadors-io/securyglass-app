# SecuryGlass App

Application de gestion pour SecuryGlass — vitrerie et miroiterie professionnelle.

## Technologies utilisées

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Développement local

Prérequis : Node.js & npm — [installer avec nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Cloner le dépôt
git clone <URL_DU_DEPOT>

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```
